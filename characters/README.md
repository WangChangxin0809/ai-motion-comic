## 技术参数

| 项目 | 说明 |
|------|------|
| 生成模型 | Doubao Seedream 4.5 (`doubao-seedream-4-5-251128`) |
| 画风 | Chinese manhua donghua style（国漫风） |
| 图片格式 | 横版四视图 (front / 3/4 / side / back)，2561×1441px PNG |
| Seed 管理 | 每角色随机 seed 已写入 JSON 档案，后续渲染自动复用 |
| 渲染脚本 | `character-design-sheet/scripts/seedream_render.py` |
| Prompt 风格 | `character design reference sheet, horizontal layout, 4 panels evenly spaced` |

## 渲染命令

```bash
export ARK_API_KEY="your-key"

# 单角色渲染
python seedream_render.py render \
  -c characters/<name>.json \
  -p "character design reference sheet, horizontal layout, ..." \
  --width 1024 --height 576 \
  -o characters/<name>_renders/turnaround.png

# seed 自动从 JSON 读取并回存，确保同角色后续出图一致
```
